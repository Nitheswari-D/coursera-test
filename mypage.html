Introduction

Trignometry is one the important branches in the history of mathematics that deals with the study of the relationship between the sides and angles of a right-angled triangle. This concept is given by the Greek Mathematician Hipparchus. In this article, we are going to learn the basics of trignometry such as trignometry functions, trignometry table, formulas and some solved examples.

What is Trignometry?

Trignometry is one the important branches in mathematics that finds huge application in diverse fields. It helps to find the missing or unknown angles or sides of a right triangle using the trignometric formulas, functions or identities. In trignometry, the angles can be either measured in degrees or radians. Trignometry is further classified into two sub branches, they are:

  • Plane Trignometry
  • Spherical Trignometry

Six Trignometric Functions

The six important trignometric functions are given below:

  1. Sine function
  2. Cosine function
  3. Tangent function
  4. Cosecant function
  5. Secant function
  6. Cotangent function

Basic Formulas

sine= opposite side / hypotenuse
tangent=opposite side / adjacent side
cosine=adjacent side / hypotenuse

Examples

Question 1:
if opp=10,hyp=5 in a right angled triangle then find the sine value?
solution
sine=10/5
sine=2
therefore sine value=2.
